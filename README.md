### Описание задачи
Компания HeadHunter хочет построить модель, которая бы автоматически определяла примерный уровень заработной платы, подходящей пользователю, исходя из информации, которую он указал о себе. Но, прежде чем построить модель, данные необходимо преобразовать, исследовать и очистить

### Описание проекта    
Проект состоит из 4 частей:
1. Исследование структуры данных
2. Преобразование данных
3. Исследование зависимостей в данных
4. Очистка данных

### Блоки выполненных практических заданий  
1. __Исследование структуры данных__
+ Чтение данных с помощью библиотеки Pandas
+ Выведим несколько первых (последних) строк таблицы
+ Выведим основную информацию о числе непустых значений в столбцах и их типах в таблице
+ Выбераем все признаки, в которых есть пропуски в данных
+ Выведим основную статистическую информацию о столбцах
2. __Преобразование данных__
+ Создан новый признак «Образование». Определяем сколько соискателей имеет средний уровень образования (школьное образование)
+ Создано два новых признака «Пол» и «Возраст». Выяснили сколько процентов женских резюме представлено в наших данных и чему равен средний возраст соискателей
+ Преобразован признак «Опыт работы» в новый признак «Опыт работы (месяц)». Определен медианный опыт работы (в месяцах) 
+ Создадали отдельные признаки «Город», «Готовность к переезду», «Готовность к командировкам»
+ Созданы признаки-«мигалки» для признаков «Занятость» и «График»
+ Создан признак «ЗП (руб)»
3. __Исследование зависимостей в данных__
+ Построено распределение признака «Возраст». Произведен анализ распределения
+ Построено распределение признака «Опыт работы (месяц)». Описано распределение
+ Построено распределение признака «ЗП (руб)». Описано распределение
+ Построена диаграмма, которая показывает зависимость медианной желаемой заработной платы («ЗП (руб)») от уровня образования («Образование»)
+ Построена диаграмма, которая показывает распределение желаемой заработной платы («ЗП (руб)») в зависимости от города («Город»)
+ Построена многоуровневая столбчатая диаграмма, которая показывает зависимость медианной заработной платы («ЗП (руб)») от признаков «Готовность к переезду» и «Готовность к командировкам» 
+ Построена сводная таблица, иллюстрирующая зависимость медианной желаемой заработной платы от возраста («Возраст») и образования («Образование»)
+ Построена диаграмма рассеяния, показывающая зависимость опыта работы («Опыт работы (месяц)») от возраста («Возраст»)
4. __Очистка данных__
+ Найдены и удалены полные дубликаты в таблице с резюме
+ Выведена информацию о числе пропусков в столбцах
+ Найдено результирующее среднее значение в столбце «Опыт работы (месяц)» после заполнения пропусков
+ Поиск выбросов
+ Поиск выбросов с помощью метода z-отклонений
